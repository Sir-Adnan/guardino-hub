# Upstream API References

This directory stores snapshots of the official upstream API contracts used while building and maintaining Guardino adapters:

- `MarzbanAPI.json`
- `PasarGuardAPI.json`
- `WGDashboard.postman_collection.v4.3.0.json`

These files are not served by Guardino and are not required at runtime. Keep them here as adapter reference material, and update the relevant snapshot before changing an adapter for a new upstream API version.
